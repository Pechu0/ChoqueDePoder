---
name: Preferencia para cambios en Unity Editor
description: Cuando el problema se puede resolver con un cambio simple en el editor de Unity, preguntar primero si el usuario quiere hacerlo él mismo
type: feedback
---

Si el problema se puede resolver agregando algo simple en el editor de Unity (ej. agregar un EventSystem, un componente, un GameObject), explicar el problema y preguntar si el usuario prefiere hacerlo él en el editor antes de editar los archivos de escena directamente.

**Why:** El usuario prefiere hacer cambios simples del editor él mismo; editar archivos .unity a mano es propenso a errores y el usuario puede hacerlo más fácilmente desde Unity.

**How to apply:** Siempre que el fix sea "agrega X en el editor de Unity", describir qué agregar y preguntar "¿quieres hacerlo tú o lo agrego al archivo?".
